## About

Design inspired by the [Monarc Framer template](https://monarc.framer.website), rebuilt as a static site.

**Font:** [Saira](https://fonts.google.com/specimen/Saira) (variable, Google Fonts)  
**Palette:** Warm neutral greys matching Monarc's exact colour tokens  
**Hosted:** GitHub Pages

## To update content

All content lives in a single `index.html` file.  
Edit the relevant HTML section and push — GitHub Pages rebuilds automatically within ~2 minutes.

```bash
# Clone
git clone https://github.com/shark79/shark79.github.io.git

# Edit index.html, then push
git add index.html
git commit -m "update: your message"
git push
```
